# TripZen

Premium tourism website scaffolded with Next.js (App Router), TypeScript, Tailwind CSS, Redux Toolkit, Framer Motion, and Lucide React.

## Local development

Install dependencies:

```bash
npm install
```

Run dev server:

```bash
npm run dev
# open http://localhost:3001 if 3000 is occupied
```

Build for production:

```bash
npm run build
npm run start
```

## Notes
- Dark glassmorphism theme
- Server/Client component separation
- Redux store in `redux/`
- Types in `types/`
